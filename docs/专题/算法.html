    # 算法

    算法是其次，主要是写码能力与熟练度。 作者：文航 链接 (opens new window) 来源：知乎

    算法虽然在平常工作中接触的可能不多，但是面试还会可能遇到，尤其是一些知名外企如微软、Facebook、Google，因为面试的人都很优秀，所以算法基础、专业技术能力、衍生技术能力都需要，但也并不是绝对。

    刷题的话建议大家直接使用VSCode安装Leetcode插件。
